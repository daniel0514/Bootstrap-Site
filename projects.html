OneClickUpload

Tired of going through every single social media application just to upload your photo? Why not let OneClickUpload do the work for you? With just one click, your photo will be instantly published to all your social media accounts! You also have the option of selecting which set of accounts to upload to! Feel like uploading the photo only to Facebook? Or think the photo should be uploaded to every social media account? With just a few clicks, OneClickUpload saves you all the trouble uploading photo one account at a time!

OneClickUpload is an Android app, written in Android Studio, to upload photos to multiple social media accounts at the same time. Currently, the app only implements two social media APIs (Facebook and Twitter). Android's built-in database, SQLite, is utilized in the app to persist user data such as Upload Profiles, preventing the loss of user created profiles between app closes. When the app is opened, the profile information is then retrieved from the database and being unmarshalled into objects for use.

Ever had the situation when you arrive at the restaurant but unable to find a parking spot nearby? You are in luck! ParkNEat will find nearby parking spots for you (however we do not guarantee the spot is empty). ParkNEat is a web application that locates restaurants near you (or a location of your choice) and display the parking spots near the restaurants of your choice. ParkNEat also allows you to see comments and rating from other users. So what are you waiting for? Start using ParkNEat

ParkNEat is developed in Java (with Eclipse IDE) and Google Web Toolkit. The web application uses Yelp's API to retrieve locations of restaurants, and parking locations are obtained from datasets from City of Vancouver Open Data Catalogue. Those data are then stored onto Google's NoSQL database for persistence. When needed, data is retrieved from the database and being displayed. Other types of data include user generated restaurant rating and reviews (not from Yelp). To display locations on a map, we integrated Google Map API and put the locations of restaurants and parking locations on the map. Twitter API is included for users to tweet about the experience with ParkNEat.

Shop Online!

Shop Online! is a website made for online shopping experience (for custoemrs) and inventory management (for managers and clerks). Customers can added items (if available) to their baskets and checkout items at once. Managers or clerks can create reports of top selling items, daily sells, and inventory counts. The website also has a login system to determine user information once accounts are created.

Shop Online is written in PHP and HTML, and the website uses MySQL database for data storage. SQL queries are specifically formulated based on user inputs and reflects exactly the behaviour we expect on the SQL server. The results of queries are then displayed in a readable format without any modification to the results For security, the website uses POST method in form handling to hide information from users. Passwords are encrypted with Blowfish Encryption and then stored in the database.

Math Riceball is an educational game that aims to promote mental math calculation. The goal of the game is to collect numbers and math operators to form math equations that results in the highest number possible. At the end, the player has to enter the answer to the math equation created to win the game.

The game was developed using Unity in Windows 10 Game Jam Vancouver 2015 by a group of four UBC students. The scripts were written in C# using Visual Studio.
